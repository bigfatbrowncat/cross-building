Инструкция по настройке окружения от bigfatbrowncat

Данная инструкция предназначена для разработки простых кроссплатформенных приложений под Windows, Linux и OS X. Она является результатом нескольких лет практики автора в его собственных проектах, поэтому если вы хотите использовать (или просто скомпилировать) один из них, вам следует начать с настройки окружения согласно этой инструкции.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages
